Windows:

To start the server manually, click on the "cgMDR Startup" icon on the desktop or choose the corresponding entry from the start menu.

You can also install the server as a service, so it will be launched automatically on startup. For this select "Install cgMDR as Service" from the start menu.

Unix/Other:

Go to the directory where you installed cgMDR and call bin/startup.sh.

Query Service Addins:

If you have chosen to include the MS Excel and/or Enterprise Architect Query Service addin, the corresponding Windows installer(s) are available at $INSTALL_PATH\cancergrid\addins. You can run the setup.exe file provided to start the addin installation process.
